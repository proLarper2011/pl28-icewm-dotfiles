# Just my custom default IceWM config. 
You can get it by yourself with:

`git clone https://github.com/proLarper2011/icewm-mydotfiles`  if you want.

The files are updated monthly if i feel for. Recommended for users who want
an minimalist and ready-to-go IceWM config.

Obs: Be aware that these files will not work if you do not place them in your
.icewm folder, on your home directory. Make sure the directory has your user own.
